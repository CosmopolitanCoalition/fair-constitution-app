About this surface

Workflows: WF-CIV-05 (candidacy lifecycle) — profile, endorsement requests, withdrawal; standings feed WF-CIV-08 (open ballot).

Entity state machine: Candidacy — Registered → Validated → In-Approval-Pool → [Endorsed] → Finalist | Non-finalist (write-in eligible) → On-Ranked-Ballot / Written-In → Elected | Not elected | Withdrawn. The strip above highlights this candidate's current state.

Deep linking: this page honors ?candidate= — every name on the open ballot links here. The count and ranked-ballot pages link non-fixture names with ?candidate=<slug>&name=<display name>&race=<election id>, which renders the same public-profile layout with the race context and an empty-but-honest record.
