Manual verification path

No external ID bridge is configured for this jurisdiction. A temporary manual path applies until in-person institutions can supplant the app-based system: an attestation session with a neutral officer of your jurisdiction stands in for the bridge. This temporary path keeps election integrity strong while in-person institutions are not yet available.
  1. Request an attestation appointment below — the queue is served by the jurisdiction's administrative office.
  2. Bring any identifying document recognized under local law.
  3. The officer records a verified flag only — no document data enters the system.

Skipping this entirely is always allowed — unverified accounts keep every right that residency grants.
