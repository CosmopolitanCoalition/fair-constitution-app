Amendments

The constitution changes through exactly two doors. Amendable variables move by ordinary legislative acts inside hardened bounds. The hardened layer itself moves only by a code release that passes the full constitutional test suite. Nothing changes silently, and nothing changes any other way.

Door two — the hardened layer

Mechanics no act, admin panel, or office can touch. A change here is a code release that must pass the full constitutional test suite before deployment — softening exists only through this door, in the open, with the diff and the test run on the public record.

The supermajority floor

The supermajority fraction is amendable, but its definition has a hardened floor: no redefinition may produce a threshold below majority + 1 of all serving members. The default is two thirds, computed as ceil(serving_members × 2/3) — against members serving, never members present.

New York County chamber: 9 serving → majority 5 · floor (majority + 1) = 6 · supermajority ceil(9 × 2/3) = 6

Supermajority, glossed: the share of every seated member — vacant seats still count in the denominator until refilled — so absence can never lower the bar.

The proportionality ratchet

The voting method (proportional voting with the Droop quota) is listed as amendable — but only in one direction: it may be replaced exclusively by a more proportional method. The engine rejects any act that would adopt plurality, first-past-the-post, or any method scoring lower on the proportionality tests.
